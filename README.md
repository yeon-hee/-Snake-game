# 🕹️ Snake_game
C++을 활용한 Snake game

### 👪 프로젝트 기간 및 참여 인원
2018.09 ~ 2018.11 팀 프로젝트(3인)

### 📄 개발 목적
- C++ 언어를 이용한 게임 개발 
- 기존 Snake game 이해 및 개선

### 🔧 사용 언어 및 라이브러리
- JAVA
- Android

### 💻 Stack
- OS : Window
- Tool : Android Studio

### 📋 구현 기능
- 회원 등록 : 이름을 입력하면 database에 멤버 추가
- 회원 정보 : database에 기록한 회원 정보 출력
- 회비 수납 : 이름과 금액을 입력 시 databse에 기록
- 미납자 : 해당 월에 대한 회비 미납자 출력
- 회비 합계 : 1년치 회비 수납 현황 출력
